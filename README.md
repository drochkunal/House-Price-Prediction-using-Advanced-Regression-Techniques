# House-Price-Prediction-using-Advanced-Regression-Techniques
Built a machine learning model to predict house prices using the Ames Housing Dataset, applying data cleaning, feature engineering, and preprocessing techniques on structured housing data (as described in data_description.txt ). Trained and compared multiple regression models including Linear Regression, Random Forest, and Gradient Boosting, and improved performance through hyperparameter tuning and cross-validation, with Gradient Boosting achieving the best results.
